Background

Groups within the Ramblers lead walks all across the UK. Details of each walk are published on the Ramblers walks finder. This facility lets you view the Meeting places and Starting places that walks have used in the past. Many of these locations (meeting and starting) may be useful for you when planning a walk.

PLEASE note that while Ramblers Groups have used these locations it DOES NOT mean that they are suitable for your use. By using this facility you are agreeing that you will check the suitability of locations before you use them.

Help

  • The location of each place is identified by the use of an Ordnance Survey Grid Reference.
  • The mapping function on the following page will show clusters of locations around the UK.
  • Hover mouse over a cluster to see its bounds and click a cluster to zoom to those bounds.
  • Click on a marker to see its Grid Reference. A list of descriptions and [when used] will be displayed.
  • You can change the type of map displayed by using the top right button.
  • If the location / grid reference is wrong then you can report this.
  • To add a new place, lead a Ramblers walk from there and get it listed on Walks Finder.
