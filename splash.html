Want somewhere to start a walk?

This facility displays places that you may wish to use to start a walk. The places are those that Ramblers walks have used in the past. Many of these will be car parks suitable for either starting a walk or for car sharing. PLEASE note that while Ramblers Groups have used these locations it DOES NOT mean that they are suitable for your use.

By clicking on the Continue button below you are agreeing that you will check the suitability of locations before you use them.

You can also view places that have been flagged as no longer suitable, by clicking on Rejected places

Help

  • The mapping function on the following page will show clusters of locations around the UK.
  • Hover mouse over a cluster to see its bounds and click a cluster to zoom to those bounds.
  • Click on a marker to see its Grid Reference. A list of descriptions and [when used] will be displayed in the panel below the map.
  • You can change the type of map displayed by using the top right button.
  • If the description or grid reference are wrong then you can report this.
  • To add a new place, lead a Ramblers walk from there and get it listed on Walks Finder
