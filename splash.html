Meeting and starting places

This facility displays the meeting and starting places that Ramblers Groups have used on their walks. It can be used to find places from which to start a walk, BUT please note that while Ramblers Groups have used these locations it DOES NOT mean that they are suitable for your use.

By clicking on the Continue button below you are agreeing that you will check the suitability of locations before you use them.

You can also view places that have been flagged as no longer suitable, by clicking on Rejected places

Mapping

  • The mapping function on the following page will show clusters of locations around the UK.
  • Hover mouse over a cluster to see its bounds and click a cluster to zoom to those bounds.
  • Click on a marker to see its Grid Reference. A list of the date used and its description will be displayed in the panel below the map.
  • You can change the type of map displayed by using the top right button.
  • If the description or grid reference are wrong then you can report this.
